Material DesignеҹәзЎҖ - Pixel Density

Posted on 2022-04-30 Edited on 2022-05-02 In frontend Views: Symbols count in article: 1.1k Reading time ≈ 1 mins.

еғҸзҙ еҜҶеәҰ(Pixel Density)

дёҖиӢұеҜёеұҸе№•дёҠеӯҳеңЁзҡ„еғҸзҙ ж•°з§°дёәеғҸзҙ еҜҶеәҰгҖӮ

еұҸе№•еғҸзҙ еҜҶеәҰе’ҢеҲҶиҫЁзҺҮеӣ е№іеҸ°иҖҢејӮгҖӮ

еұҸе№•еҜҶеәҰеҸҳеҢ–

й«ҳеҜҶеәҰеұҸе№•зҡ„жҜҸиӢұеҜёеғҸзҙ ж•°жҜ”дҪҺеҜҶеәҰеұҸе№•зҡ„еӨҡгҖӮеӣ жӯӨпјҢзӣёеҗҢеғҸзҙ е°әеҜёзҡ„UIе…ғзҙ еңЁдҪҺеҜҶеәҰеұҸе№•дёҠжҳҫзӨәеҫ—жӣҙеӨ§пјҢиҖҢеңЁй«ҳеҜҶеәҰеұҸе№•дёҠжҳҫзӨәеҫ—жӣҙе°ҸгҖӮ

и®Ўз®—еұҸе№•еҜҶеәҰ(screen density)

иҰҒи®Ўз®—еұҸе№•еҜҶеәҰпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢзӯүејҸпјҡ
еұҸе№•еҜҶеәҰ=д»ҘеғҸзҙ дёәеҚ•дҪҚзҡ„еұҸе№•е®ҪеәҰпјҲжҲ–й«ҳеәҰпјү/д»ҘиӢұеҜёдёәеҚ•дҪҚзҡ„еұҸе№•е®ҪеәҰпјҲжҲ–й«ҳеәҰпјү

еҜҶеәҰж— е…іжҖ§(Density independence)

еҜҶеәҰзӢ¬з«ӢжҖ§жҢҮзҡ„жҳҜеңЁдёҚеҗҢеҜҶеәҰзҡ„еұҸе№•дёҠз»ҹдёҖжҳҫзӨәUIе…ғзҙ гҖӮ

еҜҶеәҰж— е…ізҡ„еғҸзҙ (Density-independent pixels)

еҜҶеәҰж— е…ізҡ„еғҸзҙ пјҢеҶҷдёәdpпјҲеҸ‘йҹідёәвҖңdipsвҖқпјүпјҢжҳҜдёҖз§ҚзҒөжҙ»зҡ„еҚ•е…ғпјҢеҸҜд»ҘеңЁд»»дҪ•еұҸе№•дёҠзј©ж”ҫд»ҘиҺ·еҫ—з»ҹдёҖзҡ„е°әеҜёгҖӮе®ғд»¬жҸҗдҫӣдәҶдёҖз§ҚзҒөжҙ»зҡ„ж–№ејҸжқҘйҖӮеә”и·Ёе№іеҸ°зҡ„и®ҫи®ЎгҖӮ
Material UIдҪҝз”ЁдёҺеҜҶеәҰж— е…ізҡ„еғҸзҙ еңЁдёҚеҗҢеҜҶеәҰзҡ„еұҸе№•дёҠдёҖиҮҙең°жҳҫзӨәе…ғзҙ гҖӮ

AndroidдёҠзҡ„еғҸзҙ еҜҶеәҰ

ејҖеҸ‘Androidеә”з”ЁзЁӢеәҸж—¶пјҢдҪҝз”ЁdpеңЁдёҚеҗҢеҜҶеәҰзҡ„еұҸе№•дёҠеқҮеҢҖжҳҫзӨәе…ғзҙ гҖӮ

Dpsе’ҢеұҸе№•еҜҶеәҰ

1dpзӯүдәҺеұҸе№•еҜҶеәҰдёә160зҡ„дёҖдёӘзү©зҗҶеғҸзҙ гҖӮ
иҰҒи®Ўз®—dpпјҡ
dp=пјҲеғҸзҙ е®ҪеәҰ*160пјү/еұҸе№•еҜҶеәҰ

Screen physical width Screen density Screen width in pixels Screen width in dps
1.5 in 120 180px 240dp
1.5 in 160 240px 240dp
1.5 in 240 360px 240dp

еҸҜзј©ж”ҫеғҸзҙ пјҲScalable pixels (sp)пјү

еҸҜзј©ж”ҫеғҸзҙ пјҲspпјүзҡ„еҠҹиғҪдёҺеҜҶеәҰж— е…іеғҸзҙ пјҲdpпјүзӣёеҗҢпјҢдҪҶз”ЁдәҺеӯ—дҪ“гҖӮspзҡ„й»ҳи®ӨеҖјдёҺdpзҡ„й»ҳи®ӨеҖјзӣёеҗҢгҖӮ
spе’Ңdpд№Ӣй—ҙзҡ„дё»иҰҒеҢәеҲ«еңЁдәҺspдҝқз•ҷз”ЁжҲ·зҡ„еӯ—дҪ“и®ҫзҪ®гҖӮе…·жңүжӣҙеӨ§зҡ„еҸҜи®ҝй—®жҖ§ж–Үжң¬и®ҫзҪ®зҡ„з”ЁжҲ·е°ҶзңӢеҲ°еӯ—дҪ“еӨ§е°ҸдёҺе…¶ж–Үжң¬еӨ§е°ҸйҰ–йҖүйЎ№еҢ№й…ҚгҖӮ

iosдёҠзҡ„еғҸзҙ еҜҶеәҰ

iOSдҪҝз”ЁйҖ»иҫ‘еҲҶиҫЁзҺҮзЎ®е®ҡеҜҶеәҰпјҢиҜҘеҲҶиҫЁзҺҮд»ҘзӮ№дёәеҚ•дҪҚжөӢйҮҸеҜҶеәҰгҖӮ

йҖ»иҫ‘еҲҶиҫЁзҺҮ (Logical resolution)

еҜ№дәҺйҖ»иҫ‘еҲҶиҫЁзҺҮпјҢдҪҝз”Ёжң¬жңәжҜ”дҫӢеӣ еӯҗзј©ж”ҫзӮ№пјҢиҜҘеӣ еӯҗжҳ е°„еҲ°и®ҫеӨҮзҡ„жң¬жңәеҲҶиҫЁзҺҮпјҲд»ҘеғҸзҙ дёәеҚ•дҪҚпјүгҖӮ
дҫӢеҰӮпјҢеңЁдёәiPhone Xи®ҫи®Ўж—¶пјҢжӮЁе°Ҷи®ҫи®Ў375 X 812зӮ№зҡ„йҖ»иҫ‘еҲҶиҫЁзҺҮгҖӮжёІжҹ“ж—¶пјҢеӣҫеҪўзЎ¬д»¶дјҡеҜ№е…ғзҙ иҝӣиЎҢеӨ„зҗҶпјҢд»ҘеЎ«е……iPhone Xзҡ„1125 X 2436еғҸзҙ еұҸе№•гҖӮ

iOSзҡ„еҚ•дҪҚ

дёәiOSи®ҫи®Ўж—¶пјҢдҪҝз”ЁзӮ№пјҲptsпјү
дәҶи§ЈжӣҙеӨҡдҝЎжҒҜпјҡ
иӢ№жһңiOSеңЁжҳҫзӨәеҷЁдёҠзҡ„ж–ҮжЎЈ
PaintCodeзҡ„iPhoneеҲҶиҫЁзҺҮжҢҮеҚ—

webдёҠзҡ„еғҸзҙ еҜҶеәҰ

йҖ»иҫ‘еҲҶиҫЁзҺҮ

дҪҝз”Ёи®ҫеӨҮзҡ„йҖ»иҫ‘еҲҶиҫЁзҺҮпјҢиҜҘеҲҶиҫЁзҺҮдёҺи®ҫеӨҮзҡ„еұҸе№•еҲҶиҫЁзҺҮжҲҗжҜ”дҫӢгҖӮ

webеҚ•дҪҚ

еңЁдёәwebи®ҫи®Ўж—¶пјҢе°ҶdpжӣҝжҚўдёәpxпјҲеғҸзҙ пјү
