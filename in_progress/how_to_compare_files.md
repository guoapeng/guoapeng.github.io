## 如何使用vscode比较两个文件

1. 方法一: 使用命令行工具

  ```bash
    code -d path_to_file1 path_to_file2
  ```

2. 方法二: 使用图形节面

- 打开vscode;
- 打开两个文件，在Explorer试图中同时选中两个文件
- 右键菜单中选择，compare selected(比较选中的文件)
  
3. 方法三: 使用图形节面

- 打开vscode;
- 打开两个文件，选中一个文件, 打开右键菜单, 选择select for compare
- 选中另外一个文件, 打开右键菜单, 选择compare with selected
