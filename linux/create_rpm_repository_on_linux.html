3.5. 测试本地yum源

说明：

第一个[ ]中括号里表示仓库的名字，通常这个名字可以自己决定，以方便记忆和理解为原则，但是一般不建议里面带空格，

name：表示仓库的简要说明，

baseurl= 表示仓库的路径，也就是repodata所在的目录的父目录，

gpgcheck=0表示不检查包的gpg验证，若gpgcheck=1表示要验证。

添加以上内容我们客户端就可以根据以上定义的baseurl去服务器仓库里找相应的包。
