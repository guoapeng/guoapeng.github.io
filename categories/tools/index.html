2. 手动更新vscode

2.1. 使用 ZIP archive（文件替换）

  • 下载更新包
    点击help -> download update 系统将会自动下载更新包
  • 解压缩
    解压缩更新包
  • 替换vscode
    打开vscode的主目录, 将上一步解压缩后的文件对主目录下的文件进行替换.
  • 重启vscode, 更新完成
