1. 介绍

Visual Studio Code 简称 VS Code 是微软开发的一款跨平台的源代码编辑器。 它具有内置的调试支持，嵌入式 Git 控件，语法突出显示，代码完成，代码重构和代码片段。

Visual Studio Marketplace 提供了许多插件和扩展来扩展VS代码的功能。

本教程将指导您如何从 Microsoft 的 VS Code 存储库在Fedora 29/30上安装 VS Code 代码编辑器。
