еүҚиЁҖ

еҜ№дәҺжҲ‘д»¬иҝҷдәӣејҖеҸ‘дәәе‘ҳжқҘиҜҙпјҢеҗҜеҠЁдёҖдёӘж–°зҡ„webйЎ№зӣ®еҸҜиғҪжҳҜдёҖйЎ№д№Ҹе‘ізҡ„е·ҘдҪңгҖӮдёҖдәӣжҷ®йҖҡзҡ„еўһеҲ ж”№жҹҘйғҪиҰҒиҮӘе·ұдәІиҮӘеҠЁжүӢеҺ»еҶҷпјҢйӮЈж•ҲзҺҮзқҖе®һжҳҜеӨ§жү“жҠҳжүЈзҡ„гҖӮ

дёҖдәӣжҠҖжңҜйҖүеһӢе·ҘдҪңиҰҒдёҖж¬ЎдёҖж¬Ўең°иў«йҮҚеӨҚиҖғиҷ‘.

  • еүҚз«Ҝеә”иҜҘдҪҝз”Ёд»Җд№ҲжҠҖжңҜпјҹ
  • жҲ‘еә”иҜҘеңЁеҗҺз«ҜдҪҝз”Ёд»Җд№ҲжҠҖжңҜпјҹ
  • д»Җд№Ҳж•°жҚ®еә“жҳҜжңҖеҘҪзҡ„пјҹ

еӣ дёәзҺ°еңЁжүҖжңүзҡ„JavascriptжҠҖжңҜпјҢеҰӮReactгҖҒAngularе’ҢVueпјҢеңЁжһ„е»әWebеә”з”ЁзЁӢеәҸж—¶йғҪйқһеёёжөҒиЎҢпјҢжүҖд»ҘжҲ‘д»¬еҸҜд»ҘеҫҲеҝ«еҫ—еҲ°з¬¬дёҖдёӘй—®йўҳзҡ„зӯ”жЎҲгҖӮ
дҪҶжҳҜеҗҺз«Ҝе‘ўпјҹжҲ‘еә”иҜҘдҪҝз”ЁNodeJSиҝҳжҳҜ.NET Coreпјҹ
дҪҝз”Ёе…ізі»ж•°жҚ®еә“иҝҳжҳҜйқһе…ізі»ж•°жҚ®еә“жӣҙеҘҪпјҹ
еҗҺз«Ҝи§ЈеҶіж–№жЎҲStrapiе·Із»ҸжүҫеҲ°дәҶжүҖжңүиҝҷдәӣй—®йўҳзҡ„зӯ”жЎҲгҖӮ

еңЁжң¬ж–ҮдёӯпјҢжҲ‘д»¬е°ҶдҪҝз”ЁStrapiе’ҢAngularжһ„е»әдёҖдёӘз®ҖеҚ•зҡ„webеә”з”ЁзЁӢеәҸгҖӮ

д»Җд№ҲжҳҜStrapi?

StrapiжҳҜдёҖдёӘејҖжәҗзҡ„headlessжҺҘеҸЈдјҳе…ҲпјҲAPI-firstпјүзҡ„еҶ…е®№з®ЎзҗҶзі»з»ҹпјҢи®©ејҖеҸ‘дәәе‘ҳеҸҜд»ҘиҮӘз”ұйҖүжӢ©д»–д»¬е–ңж¬ўзҡ„е·Ҙе…·е’ҢеүҚз«ҜжЎҶжһ¶гҖӮеҮӯеҖҹдё°еҜҢзҡ„жҸ’д»¶е’Ңbuilt-inеҠҹиғҪпјҢStrapiдёәејҖеҸ‘дәәе‘ҳжҸҗдҫӣдәҶиҮӘе®ҡд№үе’ҢеҸҜжү©еұ•жҖ§зҡ„иғҪеҠӣгҖӮStrapiиҝҳжҸҗдҫӣдәҶдёҖдёӘAPIпјҢеҸҜд»ҘйҖҡиҝҮRESTжҲ–GraphQLиҪ»жқҫи®ҝй—®пјҢиҝҷеҜ№ејҖеҸ‘дәәе‘ҳйқһеёёеҸӢеҘҪгҖӮ

е®үиЈ…Strapi

иҜ·еҸӮиҖғжҲ‘зҡ„еҚҡе®ўж–Үз« е®үиЈ…strapiе’ҢдҪҝз”Ё

е®үиЈ…е®ҢstrapiеҗҺйңҖиҰҒеҲӣе»әдёҖдәӣContent-typeз”ЁдәҺжөӢиҜ•

еҲӣе»әAngularеә”з”Ё

е…ідәҺеҰӮдҪ•еҲӣе»әAngularеә”з”Ёзҡ„иҜҰз»Ҷи§ЈйҮҠеҸҜд»ҘеҸӮиҖғжҲ‘зҡ„еҚҡе®ўеҲӣе»әAngularйЎ№зӣ®.

иҝҷйҮҢеҸӘеҲ—еҮәз®ҖеҚ•жӯҘйӘӨ:

дҪҝз”Ёд»ҘдёӢе‘Ҫд»Өе®үиЈ… Angular CLIпјҡ

еҲӣе»әе№¶иҝҗиЎҢж–°зҡ„ Angular еә”з”ЁзЁӢеәҸпјҡ

еҰӮжһңжӮЁеҜјиҲӘеҲ°http://localhost:4200/жӮЁе°ҶиғҪеӨҹзңӢеҲ°ж–°зҡ„еә”з”ЁзЁӢеәҸгҖӮ

еҲӣе»әз”ЁдәҺи®ҝй—®strapiж•°жҚ®зҡ„жңҚеҠЎ

зҺ°еңЁпјҢжҲ‘д»¬еҸҜд»ҘејҖе§Ӣи®ҫи®Ўеә”з”ЁзЁӢеәҸе№¶д»ҺAPIи®ҝй—®ж•°жҚ®гҖӮйҰ–е…ҲпјҢжҲ‘д»¬е°ҶеҲӣе»әжңҚеҠЎе’ҢAPIи°ғз”Ёд»Ҙд»Һ Strapi иҺ·еҸ–ж•°жҚ®гҖӮиҝӣе…Ҙsrcж–Үд»¶еӨ№е№¶иҝҗиЎҢд»ҘдёӢе‘Ҫд»Өпјҡ

Angular CLI е°ҶеҲӣе»әpostжңҚеҠЎпјҢд»ҘдҫҝжҲ‘д»¬еҮҶеӨҮеҘҪзј–з ҒгҖӮenvironment.tsжҲ‘д»¬е°ҶеңЁе…¶дёӯиҫ“е…ҘжҲ‘д»¬зҡ„ API URLгҖӮ

еҜјиҲӘеҲ°postжңҚеҠЎе№¶жҸ’е…Ҙд»ҘдёӢд»Јз Ғпјҡ

post.service.ts

жҲ‘д»¬еҲӣе»әдәҶдёӨз§Қж–№жі•пјҡдёҖз§Қз”ЁдәҺиҺ·еҸ–жүҖжңүйЎөйқўпјҢдёҖз§Қз”ЁдәҺйҖҡиҝҮ id иҺ·еҸ–йЎөйқўгҖӮ

жіЁж„ҸпјҡдҪҝз”ЁеүҚHttpClientжҲ‘д»¬йңҖиҰҒжіЁеҶҢapp-module.ts
