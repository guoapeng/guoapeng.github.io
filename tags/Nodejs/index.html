Author: Zoltan Kochan

译者：philoenglish.com 团队

关键字： pnpm, npm, yarn

pnpm是 Node.js 的替代包管理器。它是 npm 的直接替代品，但速度更快、效率更高。

多快？快3倍！请参阅此处的基准。
